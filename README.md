# Smart City Dashboard

Real-time urban intelligence dashboard with traffic prediction using ML.

## Live Demo
https://hackathon-smart-city-dashboard.vercel.app

## Features
- Real-time weather data
- Live events from Ticketmaster
- Traffic congestion prediction (Linear Regression)
- Interactive map
- Mobile responsive

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- APIs: OpenWeatherMap, Ticketmaster, OpenAQ
- Deployment: Vercel (frontend), Render (backend)
